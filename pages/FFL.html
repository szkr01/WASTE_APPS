さよなら、退屈なループ

データの集まり（配列）全体に一度で命令。一行一行の処理を書く必要はありません。

JAVASCRIPT (合計値)

let numbers = [1,2,3,4,5];
let sum = 0;
for (let i = 0; i < numbers.length; i++) {
  sum += numbers[i];
}

ARRAY LANGUAGE (J-like)

→
15
